Welcome to telgram bot app (Alpha Smart Front GitLab)

The main endpoint is https://alpha-smart-front-gitlab.herokuapp.com/

To get chat updates the route is /get_updates [GET]

Merge request message, the route is /merge_request [POST]
Example message: tracking_front: Osman(Osman) сделал реквест на бранч develop, Последний коммит: update: Тестовый для хука
Копируй это: https://alpha-smart-front-gitlab.herokuapp.com/merge_request

Push event message, the route is /push_event [POST]
Example message: Osman(Osman) сделал пуш в реп, Количество коммитов: 4
Копируй это: https://alpha-smart-front-gitlab.herokuapp.com/push_event

Deployment event message, the route is /deployment_event [POST]
Example message: Деплой прошел
Копируй это: https://alpha-smart-front-gitlab.herokuapp.com/deployment_event
